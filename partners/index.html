パートナー統合

このパートナー統合ガイドは、以下のいずれかの条件に当てはまる場合にご利用ください。

  • 貴社がマーケティングテクノロジーベンダーであり、Tealium Customer Data Hubとの統合をご希望の場合。
  • 顧客によるTealium iQ Tag Managementでのセットアッププロセスがシームレスになるように、独自のJavaScriptタグをTealium Tag Marketplaceで提供したい場合。
  • 顧客によるTealium EventStream API HubまたはTealium AudienceStream CDPでのセットアッププロセスがシームレスになるように、独自のサーバー間APIによる統合をTealium Connector Marketplaceで提供したい場合。
  • 独自のモバイルSDKによる、Tealium for Mobileとのネイティブ統合を提供したい場合。

Tealium Customer Data Hubは、インバウンドおよびアウトバウンドのさまざまなデータ統合をサポートできるように設計されています。Tealiumのパートナーになり、自社テクノロジーを統合しましょう。

詳細については、いずれかの統合タイプをお選びください。
