CMS プラグイン

現在の管理システムにTealium Universal Tag（utag.js）を統合する方法について説明します。

Tealiumは、主要なCMSプラットフォーム用に事前ビルドされたプラグインを提供しています。プラグインはそれぞれのマーケットプレイスまたはGitHubから入手してインストールできます。Tealium CMS プラグインは、数回のクリックとわずかな構成設定でインストールできます。サイトの各ページには、Tealium Universal Tag (utag.js) と Universal Data Object (utag_data) がインストールされます。

仕組み

CMSプラグインでTealiumが正常にインストールされると、以下のパーツが実装されます。

  • コードの配置 サイトのすべてのページに、ユニバーサル タグ (utag.js) が正しく配置されます。
  • データレイヤー ビルトイン CMS ページ メタデータを使用して、 データレイヤーオブジェクト が動的かつ正確に生成されます。
  • 構成オプション アカウント名、プロファイル名、環境などの構成設定を提供します。

Tealium Wordpressプラグイン設定ガイドで、例をご確認ください。

データレイヤーバンドル

Tealium iQ Tag Managementのデータレイヤーバンドルは、CMSプラグインによって生成される、適切な変数のリストです。たとえばeコマースCMSでは、製品と注文の変数がデータレイヤーに自動的に追加されます。Tealiumユーザーは、関連するバンドルを追加することにより、これらの変数を自分のアカウント構成にすばやく追加できます。

CMSデータレイヤーバンドルの詳細については、こちらを参照してください。
