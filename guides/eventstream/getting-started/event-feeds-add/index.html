イベントフィード: フィードの追加

EventStreamのイベントフィードの追加について説明します。

次の手順は、イベントフィードを作成して、結果がゼロの検索イベントをキャプチャする方法を示していますが、この手順は、あらゆる種類のイベントフィードに使用できます。

フィードの作成

イベントフィードを作成するには、以下の手順を使用します。

  1. EventStream > Live Eventsに移動し、**+[Add Event Feed]**をクリックします。[Create Event Feed]ダイアログが表示されます。
  2. [Title]、[Notes]、および**[Labels]を構成します。ここでは[Event Data Storage]**を無視します。これを構成するには、アカウントが、DataAccessに対して有効になっている必要があります。
  3. フィードの条件を構成します。
    • ドロップダウンメニューからイベント属性を選択します。
    • 演算子を選択します。
    • 別の属性を選択するか、[Custom Value]を選択して、値を入力します。
    • 追加のAND/OR条件ロジックに対して、これを繰り返します。
  4. **[Save]**をクリックします。
  5. アカウントを保存して公開します。

Live Eventsへの再アクセス

イベント仕様とフィードを構成したので、ここでインストールに再アクセスして、EventStreamにさらにイベントを送信し、Live Eventsでどのように表示されるかを確認してみましょう。

ライブイベントとフィードをさらに理解するためのヒントをいくつか次に示します。

  • その他の属性を指定しないで"search"イベントを送信します。Live Eventsで、赤、緑、または青で表示されますか？
  • 必須属性をすべて指定して"search"イベントを送信します。Live Eventsでで有効ですか？
  • イベントフィードのLive Eventsドロップダウンで、新しいフィードを選択します。より多くのイベントをトリガーして、このビューに表示されるようにします。
