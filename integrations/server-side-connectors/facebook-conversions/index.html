Facebookのコンバージョンコネクタを

この記事ではFacebookのコンバージョンコネクタを設定する方法をご説明します。

前提条件

Facebookのコンバージョンコネクタを利用するには以下が必要です:

  • Webブラウザで動作するクライアント側の Facebookピクセルタグ。詳細は Facebookヘルプセンターをご確認ください。

FacebookピクセルをFacebookのコンバージョンコネクタと連携するには、以下の手順に従ってFacebook広告のアカウントでTealiumとのパートナー連携を有効にしてください:

  1. Facebook広告マネージャのアカウントにログインし、メニューからイベントマネージャをクリック
  2. 左側のメニューでデータソースを選択し、目的のピクセルを選択して設定タブをクリック
  3. 下にスクロールしてコンバージョンAPIセクションに移動
  4. パートナー統合経由で設定セクションでパートナーを選択をクリック
  5. Tealiumを選択後、Tealiumとのリンクを認証を有効にし、次へをクリック

パートナーを選択上でTealiumのオプションが表示されない場合は、Tealiumのアカウントマネージャにお問い合わせください。

コネクタのアクション

アクション名 AudienceStream EventStream
イベントを送信する ✓ ✓

設定

コネクタマーケットプレイスにアクセスして新しいコネクタを追加してください。コネクタを追加する手順についてはAcount Connectorsをご参照ください。

コネクタの設定タブでは以下の作業を行います:

  • ピクセルID

    • Facebookイベントマネージャから取得したピクセルIDを設定してください。
    • ピクセルを作成するには、Facebookピクセルを作成しインストールするのステップ1に従ってください。

  • テスト接続

    • テスト接続ボタンを使用してテストイベントをFacebookに送信してください。
    • ピクセルがTealium用にホワイトリスト化及び有効化されている場合、_TealiumTestConnector_イベントがピクセルを介して追跡されます。

アクション設定 - パラメータとオプション

続行をクリックするか、アクションタブに移動し、コネクタのアクションを設定してください。

このセクションでは各アクションのパラメータやオプションの設定方法をご説明します。

FacebookピクセルとFacebookコンバージョンAPIのWebイベントに対する重複除外

Facebookピクセルタグから送信されるFacebookのEvent IDは以下の命名規則に従ったイベント属性をご参照ください:

fb_event_id_<Facebookイベント>

例: fb_event_id_Purchase または fb_event_id_PageView

Event IDの種類別に個別のアクションを作成してください。イベント固有のイベントID属性をEvent IDにマッピングし、イベント名をEvent Nameにマッピングします。例えば、以下のようになります:

Tealiumの属性/値 Facebookパラメータ
fb_event_id_Purchase Event ID
購入(カスタムテキスト) Event Name

参照：Facebook Pixel Tag Setup Guide

セットアップ

FacebookとTealiumのサーバーサイドAPIパートナー統合のセットアップと管理には複数の方法があります。

  1. パートナー統合ギャラリー:
  2. データソースにおける、各ピクセルの設定タブからのパートナーを選択 ボタン:
  3. Web信号設定フローとサーバーサイドAPIガイダンスカード:

イベントの検証

イベントを送信したら、概要と内訳のビューでFacebookがイベントを受信したことを確認します:

  1. ビジネスマネージャ > ピクセルに移動
  2. POST リクエストの PIXEL_ID に紐づくピクセルをクリック

概要

概要 タブで、ビューを全て から サーバー に変更して、正常に送信されたイベントの数をご確認ください。

内訳

内訳 タブには正常に送信されたイベントの内訳が表示されます。送信に失敗したイベントは Server Events Received 列には表示されません。

サーバーからイベント送信後、20分以内にイベントを確認することができます。

検証

検証にはFacebookの Payload Helper をデータパラメータフィールドと一緒に使用し、サーバーからFacebookに送信する際のペイロードをご確認ください。問題のトラブルシューティングについてはFacebookの Support ページをご参照ください。

詳しくは、Facebookの Using the Conversions APIをご確認ください。
