Mapp Intelligence タグのセットアップガイド

この記事では、Tealium iQ Tag Management アカウントで Mapp Intelligence タグを設定する方法について説明します。

Mapp Intelligence は、ウェブサイトの利用状況の詳細な分析と貴重な洞察を提供し、顧客の深い理解を提供します。これにより、適切なメッセージを適切なタイミングで適切なチャネルに向けてターゲットオーディエンスに届けるための必要な情報が提供されます。

タグのヒント

  • データマッピングに追加された値は、Webtrekk (wt) オブジェクトに挿入されます。たとえば、wt_config.contentId や internalSearch を設定するためにデータマッピングを使用します。
  • Mapp Intelligence タグは、Form HTML ID (formTrackInstall) に値を設定することで、formTrackInstall() を自動的にトリガーします。
  • イベントトラッキングは、utag.link() を使用して関数呼び出しで行うことができます。
  • オーダーの値を自動的に設定するには、E-Commerce 拡張機能が必要です。

タグの設定

新しいタグを追加するために、タグマーケットプレイスに移動します。タグの追加方法の一般的な手順については、タグの概要 の記事を参照してください。

タグを追加する際に、以下の設定を構成します：

  • ライブラリのバージョン: 使用しているライブラリのバージョン。
  • トラック ID: リクエストをアカウントに割り当てるために使用される ID。Mapp システム設定のデータ収集に移動して、トラック ID を確認します。同じ情報を複数のアカウントに記録する必要がある場合は、アカウントの対応するトラック ID をカンマ区切りのリストとして入力します。
  • トラックドメイン: アカウントマネージャーから提供された Mapp Intelligence トラックドメイン。
  • 最初のパラメータ: 送信される最初のパラメータのセミコロン区切りのリスト。
  • ドメイン: 追跡されるウェブサイトのドメイン。セミコロンで区切って複数のドメインを入力できます。
  • リンクトラッキング: Mapp Intelligence タグには、HTML リンクをトラッキングするための2つの方法があります：
    • リンク: クリックされたときにすべての HTML リンクが自動的にトラッキングされます。
    • スタンダード: ページ内のすべてのラベル付き HTML リンクがクリックされたときにトラッキングされます。
  • リンクトラック属性: リンク名に使用される HTML 属性の名前。
  • リンクトラックパラメータ: リンク URL のクエリ文字列パラメータの名前。リンク名として使用されます。
  • リンクトラックダウンロード: トラッキングするダウンロードファイルの拡張子。たとえば、.zip、.rar、.doc
  • ピクセルサンプリング: ページ上でトラッキングするユーザーを n 回に1回だけにするかどうかを定義します。
  • 強制的な HTTPS: オン または オフ を選択します。
  • クッキーハンドリング: 使用するクッキーを選択します。ファーストパーティクッキーまたはサードパーティクッキー。
  • クッキードメイン
  • mediaCode: 標準キャンペーントラッキングに使用するパラメータとして mediaCode を設定します。
  • mediaCodeCookie: アドレスバーに mediaCode クエリパラメータがある場合（たとえば、AJAX で動作するウェブサイトの場合）、mediaCodeCookie を sid に設定します。
  • WT オブジェクト名: WT オブジェクトの名前。単一の Webtrekk インスタンスを使用する場合は、デフォルトの wt のままにしておきます。
  • フォームトラッキング: グローバルなフォームトラッキングを有効にします。単一のページでフォームトラッキングを有効にするには、データマッピングで Form HTML ID を使用します。
  • リクエストキュー: オン または オフ を選択します。
  • サーバートゥサーバートラッキング: オン または オフ を選択します。

ロードルール

タグをすべてのページで読み込むか、タグが読み込まれる条件を設定します。ロードルールについての詳細は、ロードルール のドキュメントを参照してください。
