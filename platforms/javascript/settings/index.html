JAVASCRIPT

設定

utag.jsの動作を調整するために使用可能な設定について説明します。

仕組み

utag.jsの動作の多くは、utag.cfgオブジェクト内の設定によって制御されます。これらの設定によって制御されるデフォルトの動作は、utag_cfg_ovrdという名前の新しいオブジェクトを使用してオーバーライドされます。このオブジェクトを使用するには、utag.jsを読み込む前にページコードで設定するか、または[Pre Loader]をスコープとするJavaScriptコード拡張機能内で以下のように設定します。

window.utag_cfg_ovrd = window.utag_cfg_ovrd || {};

設定

以下の表は、利用可能な設定をまとめたものです。

Setting 説明
cmcookiens 同意管理 Cookie の名前。
consentPeriod ユーザーの同意設定を保持する日数を設定します。
dom_complete DOM 完了時 (読み込み) イベントまでタグを遅延します。
domain Cookie を設定するために用いられるドメインをオーバーライドします。
gdprDLRef 同意マネージャーによって使用される言語設定を保存するデータレイヤー変数の名前を指定します。
load_rules_ajax ページ読み込み後に読み込みルールを無効化します（レガシー）。
load_rules_at_wait 拡張機能後に読み込みルールを実行します（レガシー）。
lowermeta Lower-case meta tag names and values (legacy).
lowerqp Lower-case query string parameter names and values (legacy).
noload すべての機能を無効化します。
noview 最初のページ読み込み時における自動トラッキング コールを無効化します。
nocookie utag_main Cookie を無効化します。
readywait DOM 対応ブラウザ イベントまで動作を停止します。
secure_cookie 属性文字列を secure に設定します (utag_main Cookie で、 Persist Data Values (永続性データ値) 拡張機能 と SC() 関数によって設定されたものすべてのために)。
session_timeout Set the session expiration time (in milliseconds).
waittimer タグを読み込む前の遅延期間 (ミリ秒単位) を設定します。

cmcookiens

Consent Management (同意管理) クッキーの名前 Cookie の名前をカスタマイズします。異なる複数のプロファイルで同一のドメイン名を共有しているときに、これは有用です。

window.utag_cfg_ovrd.cmcookiens = "CONSENTMGR_NL-CMB-CG1";

consentPeriod

同意設定の保持期間 GDPR および CCPA に関する同意状態を保持する日数を設定します。ビルトインの有効期限は、GDPR が 365 日、CCPA が 395 日です。

window.utag_cfg_ovrd.consentPeriod = 60;

dom_complete

DOM 完了時 (読み込み) ブラウザ イベントまでタグを遅延します DOM 対応時ではなく ドキュメント読み込みイベント 時にタグを読み込みます。 また、[DOM Ready] をスコープとするすべての拡張機能を抑制します。 (デフォルト: false)

window.utag_cfg_ovrd.dom_complete = true;

domain

Cookieドメインのオーバーライド utag_main Cookieが設定されるドメインを設定します。Cookieを受け付けないルートドメイン上のサイトに対して役立ちます。たとえば、amazonaws.comなどです。（デフォルト値location.hostnameの最上位ドメイン）

window.utag_cfg_ovrd.domain = "mysite.amazonaws.com";

gdprDLRef

同意マネージャーの言語設定をオーバーライドします 同意マネージャーで使用される言語設定を保存するデータレイヤー変数の名前を指定します。

たとえば、データレイヤーに、 site_languageという名前の変数が含まれる場合は:

window.utag_cfg_ovrd = window.utag_cfg_ovrd || {}
window.utag_cfg_ovrd.gdprDLRef = "site_language";

言語コードを直接設定しません。このオーバーライド設定では、変数名を想定しており、言語コード値を想定していません。

load_rules_ajax

ページ読み込み後にタグ配信ルールを無効化（レガシー） 同じページ読み込み内で utag.view および utag.link を呼び出すたびに読み込みルールを再処理するかどうかを制御します。新しくトリガーされたタグ配信ルールにより、ページに新しいタグが読み込まれる可能性があります。この設定は、4.2xより古いバージョンのutag.jsでの動作を再現する場合に使用します。（デフォルト：true）

window.utag_cfg_ovrd.load_rules_ajax = false;

load_rules_at_wait

拡張機能後にタグ配信ルールを実行（レガシー） 拡張機能後にタグ配信ルールを評価します。古いバージョンのutag.jsで、utag.jsの読み込み後にデータレイヤーオブジェクトに値が入力されるようなインストール環境に対して使用されます。新しいバージョンのutag.jsでは、この設定の代わりに拡張機能の実行順序を利用できます。（デフォルト：false）

window.utag_cfg_ovrd.load_rules_at_wait = true;

lowermeta

小文字のメタタグデータ（レガシー） メタデータ変数の名前と値をすべて小文字にします。この設定は、4.2xより前のバージョンのutag.jsでの動作を再現します。（デフォルト：false）

メタ タグの例:

<meta content="iQ Tag Management" property="Article:Section">

結果の値:

utag.data['meta.article:section']="iq tag management"
window.utag_cfg_ovrd.lowermeta = true;

lowerqp

小文字のクエリ文字列パラメータの名前と値 (レガシー) クエリ文字列の変数の名前と値をすべて小文字にします。この設定は、4.2xより前のバージョンのutag.jsでの動作を再現します。（デフォルト：false）

クエリ文字列パラメータの例: &RefId=Abc123 結果の値: utag.data['qp.refid']="abc123"

window.utag_cfg_ovrd.lowerqp = true;

noload

すべての動作の停止 Pre Loaderをスコープとする拡張機能の後に、すべてのコードの実行が停止します。[DOM Ready] をスコープとする拡張機能は、引き続き実行されます。これは通常、 [Prevent Tag Load (タグの読み込みを防止する)]の公開設定を使用して調整されます。（デフォルト：0）

window.utag_cfg_ovrd.noload = true;

noview

最初のページ読み込み時における自動トラッキング コールを無効化します 最初のページ読み込み時に自動的になされるトラッキング コールを抑制します。この設定は、一般に、単一ページ アプリケーション サイトで使用されます。（デフォルト：false）

window.utag_cfg_ovrd.noview = true;

nocookie

Cookieのオプトアウト このオプションは、訪問者が明示的にすべての Cookie をオプトアウトした場合のみ設定します。（デフォルト：false）

このオプションを使用すると、訪問者およびセッションの数が増加し、すべての訪問が単一ページセッションのように見えます。

このオプションを設定すると、 Persist Data Value (永続性データ値) 拡張機能で設定された Cookie やセッション Cookie を含めて、 すべて の Cookie が utag.jsによって保存されなくなります。また、Cookieが使用できないとき、変数ut.visitor_id、tealium_visitor_id、cp.utag_main_v_idに新しいタイムスタンプが設定されます。

window.utag_cfg_ovrd.nocookie = true

readywait

DOM 対応までの動作の遅延 ブラウザから DOM 対応信号が受信されるまで、すべての動作を停止します。[Pre Loader]をスコープとする拡張機能は、utag.jsが読み込まれても引き続き実行されます。（デフォルト：0）

window.utag_cfg_ovrd.readywait = true;

secure_cookie

セキュアな Cookie 属性文字列を secure に設定します (utag_main Cookie で、 Persist Data Values (永続性データ値) 拡張機能 と SC() 関数によって設定されたものすべてのために)。セキュアな Cookie の設定とアクセスは、HTTPS ページ上でのみ可能です。（デフォルト：false）

utag.js バージョン 4.48 が必須です

window.utag_cfg_ovrd.secure_cookie = true;

session_timeout

セッションのタイムアウト 現在のセッションを有効期限切れにするまで待つ時間（ミリ秒単位）を設定します。これは通常、[Session Timeout (セッションのタイムアウト)] の公開設定を使用して調整されます。（デフォルト値：1800000（30分））

タイムアウトの設定の例としては、900000 ミリ秒 (15 分):

window.utag_cfg_ovrd.session_timeout = 900000;

waittimer

タイマによるタグの遅延 DOM対応イベント後にタグを読み込むまで待つ時間（ミリ秒単位）を設定します。（デフォルト：設定なし）

window.utag_cfg_ovrd.waittimer = 1000;
