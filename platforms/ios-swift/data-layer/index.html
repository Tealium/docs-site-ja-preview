データレイヤー

各モジュールによって提供される、共通データレイヤー値とデータの管理について説明します。

カスタム データ

各イベントにおいて一般的なデータレイヤー値が必要になるのは一般的なことです。各トラッキング コールにこうした値を追加する必要性を回避するためには、すべてのトラッキング コールに適用する永続性データレイヤー値に tealium.dataLayer メソッドを用います。値が保存され次第、これらの値は、トラッキング対象の各イベントに自動的に含められます。

データの有効期限

データレイヤー値には、有効期限があり、有効期限が過ぎるとデータレイヤーから削除されます。カスタム データレイヤー値を設定すると、任意で、値の有効期限を設定できます。デフォルト有効期限は、現在のセッション中です。

使用可能な有効期限のオプションは、次のとおりです。

Type 値 説明
セッション中 (デフォルト) .session 現在のセッションの終了時に有効期限が切れます
再起動時 .untilRestart アプリがクローズされたまたは再起動されたときに有効期限が切れます
永久 .forever 有効期限が決して切れません
カスタム日付の時点 .after 指定日に有効期限が切れます
カスタム期間中 .afterCustom 次の時間単位を使って指定した期間が経過した時点で有効期限が切れます: .minutes、 .hours、 .days、 .months、 .years

add() を用いて、有効期限のタイプに基づき、データレイヤーにデータを保存します。

セッション データ

セッション データ値は、セッションの間、永続化されます。セッションは、所与のタイム フレーム内でのユーザー アクティビティによって決定されます。セッションは、ユーザーのインアクティビティが 30 分間継続した後に有効期限が切れます。

セッションの後に有効期限が切れたデータレイヤー値を永続化するには:

tealium?.dataLayer.add(key: "mysessionvar", value: "hello", expiry: .session)

すべてのセッション データを取得するには:

let sessionData = tealium?.dataLayer.allSessionData

再起動データ

再起動データ値は、各イベントの間、永続化されます。アプリがクローズされたまたは再起動されたときに、データが削除されます。

次のアプリの再起動までデータレイヤー値を永続化するには:

tealium?.dataLayer.add(key: "myrestartvar", value: "foo", expiry: .untilRestart)

永久データ

永久データ値は、現在のアプリ バージョンのライフタイムの間、永続化されます。アプリがアンインストールされたまたは更新されたときに、データレイヤーからデータが削除されます。

永久データを永続化するには:

tealium?.dataLayer.add(key: "myforevervar", value: "world", expiry: .forever)

カスタム日付データ

カスタム データ日付値は、指定日まで永続化されます。

カスタム日付の有効期限を有するデータレイヤー値を永続化するには:

tealium?.dataLayer.add(key: "customdate", value: "bar", expiry: .after(yourDate))

カスタム期間データ

カスタム期間データ値は、指定期間によって永続化されます。

  1. 日経過後に有効期限が切れるデータレイヤー値を永続化するするには:

    tealium?.dataLayer.add(key: "lengthoftime", value: "days", expiry: .afterCustom((.days, 1)))

  2. 月経過後に有効期限が切れるデータレイヤー値に永続性を付与するには:

    tealium?.dataLayer.add(key: "lengthoftime", value: "months", expiry: .afterCustom((.months, 1)))

全データの取得

データレイヤーに現在保存されているすべてのデータを取得するには、 all() メソッドを使用します。

let data: [String: Any] = tealium?.dataLayer.all

Tealium.gatherTrackData()も参照してください。

データのクリア

特定のキーのデータをクリアするには、 メソッドを使用し、削除するには () メソッドを使用します。

  • 複数: dataLayer.delete(forKeys:)
  • 単一: dataLayer.delete(forKey:)

すべてのデータをクリアするには、 deleteAll() メソッドを使用します。

例

次の例は、上記のメソッドを示しています。

class TealiumHelper {
	var tealium: Tealium?
	let config = TealiumConfig(...)
	tealium = Tealium(config: config) { [weak self] _ in
		guard let self = self, let teal = self.tealium else { return }

		teal.dataLayer.add(key: "mysessionvar", value: 123456)

		teal.dataLayer.add(key: "myvarforever", value: "hello", expiry: .forever)

		teal.dataLayer.add(key: "campaign", value: "summer", expiry: .afterCustom((.months, 3)))

		print("All event data: \(teal.dataLayer.all)")
		print("All session data: \(teal.dataLayer.allSessionData)")
		print("Single event data key `campaign`: \(teal.dataLayer.all["campaign"])")

		teal.dataLayer.delete(forKeys: ["myvarforever"])

		teal.dataLayer.deleteAll()
	}
}
