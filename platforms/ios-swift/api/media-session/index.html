IOS SWIFT

MediaSession

Tealium for iOS（Swift）で提供されている Media のクラスとメソッドに関するリファレンスガイド。

クラス: メディア

MediaSession クラスは、カスタム イベントの使用を通じてアプリ内のストリーミング メディアを追跡するメソッドを提供します。メディア トラッキングの詳細は、こちらから参照してください。

以下は、一般的に使用される Tealium iOS (Swift) 向けの MediaSession クラスのメソッドとプロパティをまとめたものです。

メソッド or Property 説明
bitrate Quality of Experience (QoE) ビットレート値を設定します。
clickAd() ユーザーが、現在表示されている広告をクリックしました。
custom() カスタム イベント ステータスをデータレイヤーに送信します。
endAd() 広告が終了しました。
endAdBreak() 広告ブレークが終了しました。
endBuffer() バッファが終了しました。
endChapter() チャプターが終了しました。
endSeek() ユーザーがシークを終了しました。
endSession() 広告とコンテンツを含むセッションが終了しました。
pause() ユーザーがメディアを一時停止しました。
ping() インターバル イベントをデータレイヤーに送信します。
play() メディア コンテンツが再生を開始しました。
playerState プレイヤーの開始または終了の状態。
resumeSession() セッションが再開しました。
sendMilestone() マイルストーン データをデータレイヤーに送信します。
sendSummary() サマリー データをデータレイヤーに送信します。
skipAd() ユーザーが現在の広告をスキップしました。
skipChapter() ユーザーがチャプターをスキップしました。
startAdBreak() 広告ブレークが開始しました。
startAd() 広告が開始しました。
startBuffer() バッファが開始しました。
startChapter() チャプターが開始しました。
startSeek() ユーザーがシークを開始しました。
startSession() セッションが開始しました。

bitrate()

現在セッションの Quality of Experience (QoE) ビットレート値 (Int) (キロバイト/秒 (kpps) 単位) を設定します。

Property 型 説明
bitrate Int ビットレート値。 
mediaSession.bitrate = BITRATE_VALUE

clickAd()

ユーザーが、現在表示されている広告をクリックしました。

mediaSession.clickAd()

custom()

スタム イベント ステータスをデータレイヤーに送信します。

mediaSession.custom(name)
パラメータ 型 説明
name String カスタム イベント名:

endAd()

広告が終了しました。

mediaSession.endAd()

endAdBreak()

広告ブレークが終了しました。

mediaSession.endAdBreak()

endBuffer()

バッファが終了しました。

mediaSession.endBuffer()

endChapter()

チャプターが終了しました。

mediaSession.endChapter()

endSeek()

ユーザーがシークを終了しました。シークが所与のポジションで終了したことを記録します。

mediaSession.endSeek(at position: Int)
パラメータ 型 説明
position Int シークが終了したメディアのインデックス ポジション (秒単位)。

例:

mediaSession.endSeek(at position: 10)

endSession()

広告とコンテンツを含むセッションが終了しました。

mediaSession.endSession()

pause()

ユーザーがメディアを一時停止しました。

mediaSession.pause()

ping()

インターバル イベントをデータレイヤーに送信します。このメソッドは、 MediaContent オブジェクト用のトラッキング タイプが interval または intervalMilestoneに設定された場合に 10 秒ごとに自動的に呼び出されます。

mediaSession.ping()

メディア トラッキングのタイプの詳細は、こちらから参照してください。

play()

メディア コンテンツが再生を開始しました。

mediaSession.play()

playerstate

プレイヤーの開始または終了の状態。

mediaSession.playerState = PLAYER_STATE

以下の State 定数を利用できます。

値 説明
closedCaption 字幕状態のタイプ。
fullscreen 全画面状態のタイプ。
inFocus イン フォーカス状態のタイプ。
mute ミュート状態のタイプ。
pictureInPicture ピクチャー イン ピクチャー状態のタイプ。

例:

mediaSession.playerState = .mute

プレイヤーの状態の詳細は、こちらから参照してください。

resumeSession()

セッションが再開しました。

mediaSession.resumeSession()

sendMilestone()

マイルストーン イベントをデータレイヤーに送信します。このメソッドは、 MediaContent オブジェクト用のトラッキング タイプが milestone または intervalMilestoneに設定された場合にコンテンツの再生された割合 (%) に基づき自動的に呼び出されます。

mediaSession.sendMilestone(_ milestone: Milestone)
パラメータ 型 説明
milestone Milestone コンテンツの再生された割合 (%)

マイルストーン値:

Media % Played マイルストーン値
8.0 - 12.0 10%
23.0 - 27.0 25%
48.0 - 52.0 50%
73.0 - 77.0 75%
88.0 - 92.0 90%
97.0 - 100.0 100%

メディア トラッキングのタイプの詳細は、こちらから参照してください。

sendSummary()

サマリー データをデータレイヤーに送信します。このメソッドは、 endSession() に基づき、 MediaContent オブジェクト用のトラッキング タイプが summaryに設定された場合に自動的に呼び出されます。

mediaSession.sendSummary()

メディア トラッキングのタイプの詳細は、こちらから参照してください。

skipAd()

ユーザーが現在の広告をスキップしました。

mediaSession.skipAd()

skipChapter()

ユーザーがチャプターをスキップしました。

mediaSession.skipChapter()

startAd()

広告が開始しました。広告が完了したときに endAd() メソッドを呼び出します。

mediaSession.startAd(_ ad: Ad)
パラメータ 型 説明
ad Ad 広告オブジェクト。

例:

let ad = Ad(name = "Ad 1")
session.startAd(ad)

startAdBreak()

広告ブレークが開始しました。広告ブレークが完了したときに endAdBreak() メソッドを呼び出します。

mediaSession.startAdBreak(_ break: AdBreak)
パラメータ 型 説明
break AdBreak 広告ブレーク オブジェクト

例:

let adBreak = AdBreak(name = "Ad Break 1")
mediaSession.startAdBreak(adBreak)

startBuffer()

バッファが開始しました。バッファが完了したときに endBuffer() メソッドを呼び出します。

mediaSession.startBuffer()

startChapter()

チャプターが開始しました。チャプターが完了したときに endChapter() メソッドを呼び出します。

mediaSession.startChapter(_ chapter: Chapter)
パラメータ 型 説明
chapter Chapter チャプター セグメント名

例:

let chapterOne = Chapter(name = "Chapter 1")
mediaSession.startChapter(chapterOne)

startSeek()

ユーザーがシークを開始しました。シークが完了したときに endSeek() メソッドを呼び出します。

mediaSession.startSeek(at position: Int)
パラメータ 型 説明
position Int シークが開始したメディアのインデックス ポジション (秒単位)。

例:

mediaSession.startSeek(at position: 10)

startSession()

セッションが開始しました。セッションが完了したときに endSession() メソッドを呼び出します。

mediaSession.startSession()

クラス: MediaContent

MediaContent クラスは、メディア コンテンツ オブジェクトを作成します。

let media = MediaContent(
  uuid: MEDIA_UUID,
  name: MEDIA_NAME,
  streamType: MEDIA_STREAM_TYPE,
  mediaType: MEDIA_TYPE,   
  qoe: MEDIA_QOE,
  trackingType: MEDIA_TRACKING_TYPE,
  milestoneInterval: MEDIA_MILESTONE,
  contentCompletePercentage: MEDIA_CONTENT_COMPLETE_PERCENTAGE,
  startTime = MEDIA_START_TIME,
  state = MEDIA_STATE,
  customID = MEDIA_CUSTOM_ID,
  duration: MEDIA_DURATION,
  playerName: MEDIA_PLAYER_NAME,
  channelName: MEDIA_CHANNEL_NAME,
  metadata: MEDIA_METADATA,
  milestone: MEDIA_MILESTONE,
  summary: MEDIA_SUMMARY,
  adBreaks: MEDIA_AD_BREAKS,
  ads: MEDIA_ADS,
  chapters: MEDIA_CHAPTERS
)
変数名 型 説明 必須
uuid UUID 広告ユニバーサル一意識別子。
name String メディア オブジェクトの名前。 はい
streamType StreamType メディア ストリームの名前。 はい
mediaType MediaType メディア ストリームのタイプ (ビデオまたはオーディオ)。 はい
qoe QoE Quality of Experience (QoE) ビットレート値。 はい
trackingType TrackingType イベント トラッキングのレベル。(デフォルト: .fullPlayback) はい
milestoneInterval Double マイルストーン インターバル イベント。(デフォルト: 5.0)
contentCompletePercentage Double コンテンツの完了割合 (%) 値。
state PlayerState メディア プレイヤー状態のタイプ。
customId String カスタム識別子名。
duration Int メディア期間 (秒単位)。たとえば 130
playerName String メディア プレイヤーの名前。
channelName String メディア チャンネルの名前。
metadata [文字列: 任意] Additional media メタデータ。
summary Summary メディア Summary オブジェクト。
adBreaks [AdBreak]() AdBreak オブジェクトの配列。
ads [Ad]() Ad オブジェクトの配列。
chapters [Chapter]() Chapter オブジェクトの配列。

構造体: Ad

Ad 構造体は、広告オブジェクトを作成します。

let ad = Ad(
  uuid: MEDIA_AD_UUID,
  name: MEDIA_AD_NAME,
  id: MEDIA_AD_ID,
  duration: MEDIA_AD_DURATION,
  position: MEDIA_AD_POSITION,
  advertiser: MEDIA_AD_ADVERTISER,
  creativeId: MEDIA_AD_CREATIVE_ID,
  campaignId: MEDIA_AD_CAMPAIGN_ID,
  placementId: MEDIA_AD_PLACEMENT_ID,
  siteId: MEDIA_AD_SITE_ID,
  creativeUrl: MEDIA_AD_CREATIVE_URL,
  numberOfLoads: MEDIA_AD_LOAD,
  pod: MEDIA_AD_POD,
  playerName: MEDIA_AD_PLAYER_NAME
)
パラメータ 型 説明 必須
uuid UUID 広告ユニバーサル一意識別子。
name String 広告名。 はい
id String 広告一意識別子。
duration Double 広告期間 (秒単位)。
position Int 広告の現在のインデックス ポジション (秒単位)。
advertiser String 広告の広告主。
creativeId String 広告クリエイティブ識別子。
campaignId String 広告キャンペーン識別子。
placementId String 広告プレースメント識別子。
siteId String 広告サイト識別子。
creativeUrl String 広告クリエイティブ URL。
numberOfLoads Int 広告読み込みの数。
pod String 広告ポッド。
playerName String 広告プレイヤーの名前。

構造体: AdBreak

AdBreak 構造体は、広告ブレーク オブジェクトを作成します。

let adBreak = AdBreak(
  uuid: MEDIA_AD_BREAK_UUID,
  name: MEDIA_AD_BREAK_NAME,
  id: MEDIA_AD_BREAK_ID,
  duration: MEDIA_AD_BREAK_DURATION,
  position: MEDIA_AD_BREAK_POSITION,
  startTime: MEDIA_AD_BREAK_START_TIME
)
パラメータ 型 説明 必須
uuid UUID 広告ブレーク ユニバーサル一意識別子。
name String 広告ブレークの名前。 はい
id String 広告ブレーク一意識別子。

構造体: Chapter

Chapter 構造体は、チャプター オブジェクトを作成します。

let chapterObject = Chapter(
  name: MEDIA_CHAPTER_NAME,
  duration: MEDIA_CHAPTER_DURATION,
  position: MEDIA_CHAPTER_POSITION,
  startTime: MEDIA_CHAPTER_START_TIME,
  metadata: MEDIA_CHAPTER_METADATA
)

Chapter オブジェクト パラメータは、次のとおりです。オブジェクトの作成時に duration: などのパラメーター名を用いない場合には、パラメータは整理されている必要があります。
