JAVA

インストール

イベントのアクティビティをトラッキングするために、TealiumをJavaアプリケーションに追加します。

要件

  • Tealium Universal Data Hubアカウント
  • Java IDE、例： Eclipse
  • Java 7以上

インストール

Tealium Javaライブラリのコードは、GitHubで公開しています。

*ダウンロード：Tealium for Java

今後のリリースに向けたアップデートをしやすくするため、ライブラリを（ダウンロードではなく）クローンすることを推奨しています。

src/main/java/com/tealiumディレクトリからすべての.javaファイルをインポートし、以下のコードを追加します。

import com.tealium.*

Mavenの構成

pom.xmlファイルに以下の変更を加えます。

レポジトリを追加します。

<repository>    
  <id>maven-tealium</id>
  <url>http://maven.tealiumiq.com/java/releases/</url>
</repository>

依存関係を追加します。

<groupId>com.tealium</groupId>
<artifactId>java</artifactId>
<version>1.3.0</version>

初期化

Tealiumインスタンスを初期化します。

Tealium tealium = new Tealium.Builder("ACCOUNT", "PROFILE")
    .setEnvironment("ENVIRONMENT")
    .setDatasource("DATASOURCE")
    .build();
パラメータ 詳細
account アカウント名
profile プロファイル名
environment “dev”、”qa”、”prod”のいずれか
datasource （任意）UDHのデータソースキー（ない場合はnullに設定）

トラッキング

イベントのトラッキングには、track()メソッドを使用します。

// Set event attributes
Udo data = new Udo();
data.put("some_key", "some_value");

// With optional callback
DispatchCallBack callback = new DispatchCallBack(){
    public void dispatchComplete(boolean success, String encodedURLString, String error){
       // Callback handling
       System.out.println("Dispatch successful: " + String.valueOf(success));    
   }
}

// Tracking a login event
tealium.track("event", data, callback);
パラメータ 詳細
event （必須）イベントを特定する名前
data （任意）キーと値のペアとしてイベントデータを持つオブジェクト
callback （任意）”callback”キーに割り当てられた関数を持つオブジェクト
