TealiumConfig

メインのTealiumクラスの構成オプションを設定するためのクラスです。個々のモジュールでは、 TealiumConfig クラスにモジュール独自の拡張機能を提供します (有効な場合)。

TealiumConfig

以下は、 TealiumConfig クラスのプロパティをまとめたものです。

パラメータ 型 説明 例
account String (必須) Tealium アカウントの名前 companyXYZ
collectors List<Collectors> (必須) 次を用いて Tealium ライブラリを初期化するための Collectors のリストを設定します [Collectors.AppData]
consentExpiry ConsentExpiry ユーザーの同意設定の有効期限を設定します (デフォルトは、ポリシーによります) ConsentExpiry(90, TimeUnit.DAYS)
consentLoggingEnabled Boolean 同意の記録機能を有効にします。この機能では、監査目的で同意ステータスの変化をすべてTealium Customer Data Hubに送信します。(デフォルト: 有効) true
consentPolicy ConsentPolicy 同意ポリシー (CCPA または GDPR など) を設定します。Consent Manager は、このプロパティが設定されている場合のみ有効です。 ConsentPolicy.GDPR
customVisitorId String カスタム訪問者 ID を設定します “ALK2398LSDKJ3289SLKJ3298SLKJ3”
dataSource String CDH データ ソース キー "abc123"
deepLinkTrackingEnabled Boolean 標準ディープ リンクの自動トラッキング (Facebook などのソースからアプリへのリンクや、QR トレースなど) を有効化または無効化します。(デフォルト: 有効) false
dispatchers Dispatchers[] (Required) Sets the list of Dispatchers 次を用いて Tealium ライブラリを初期化するには [Dispatchers.Collect]
environment TealiumEnvironment (必須) Tealium 環境の名前 TealiumEnvironment.dev
lifecycleAutotrackingEnabled Boolean ライフサイクル自動トラッキングを有効化または無効化します。(デフォルト: 有効) false
logLevel LogLevel` Sets the log level property, which controls how much information is logged (default: silent) LogLevel.DEV
memoryReportingEnabled Boolean DeviceData モジュールでメモリ レポートを有効または無効にします (デフォルトでは無効) true
overrideCollectBatchURL String データを別のエンドポイントに送信するように Tealium Collect のバッチ URL をオーバーライドします “https://custom-domain.com/batch-event”
overrideCollectURL String データを別のエンドポイントに送信するようにTealium CollectのURLをオーバーライドします。イベント バッチング機能を用いる場合は、 overrideCollectBatchURL プロパティのオーバーライドもします。 “https://custom-domain.com/event”
overrideLibrarySettingsURL String 公開設定の URL をオーバーライドします。` “https://custom-domain.com/mobile.html”
overrideTagManagementURL String Tag Managementモジュールで使用されるデフォルトのURLをオーバーライドします。これは、Tealium perlScript ファイルをセルフホストしている場合に必須です “https://custom-domain.com/path/env/utag.js”
profile String (必須) Tealium プロファイルの名前 main
qrTraceEnabled Boolean QR トレースを有効化または無効化します。(デフォルト: 有効) false
useRemoteLibrarySettings Boolean モバイル公開設定 (デフォルト: 有効) を有効化または無効化します。モバイル公開設定を iQ Tag Management で設定するか、機能を無効化します。 false
visitorServiceEnabled Boolean データレイヤーエンリッチメント API を使用して Visitor Profile の自動取得を有効化または無効化します (デフォルト: 無効) true

Collectors

コレクター (Collectors) は、デバイスから補足情報を収集し、データレイヤーにそれを添付して、Tealium Customer Data Hub にそれを転送するモジュールです。コレクターの中にはコア ライブラリに含まれるものもあれば、オプションで、個別のモジュールとしてインストールされるものもあります。

以下の表では、利用可能なコレクターを列挙しています。

Collector Name TealiumConfig のリファレンス
AppData (default) Collectors.AppData
Connectivity (default) Collectors.Connectivity
Device Collectors.Device
Lifecycle Collectors.Lifecycle

このモジュールは、 TealiumConfig collectors プロパティを用いて有効化または無効化されます。

ConsentExpiry

ユーザー同意設定の有効期限を定義します。

パラメータ 型 説明 例
time Number 有効期限が満了するまでの期間 90
unit TimeUnit 有効期限が満了するまでの期間の単位 TimeUnit.DAYS

例:

new ConsentExpiry(90, TimeUnit.Days)

以下の期間の単位を利用できます。

Time Unit 説明
ConsentExpiry.DAYS 日
ConsentExpiry.HOURS 時間
ConsentExpiry.MINUTES 分
ConsentExpiry.MONTHS 月

ConsentPolicy

遵守するべき同意ポリシーを定義します。同意ポリシーが TealiumConfig オブジェクトについて定義されていない場合は、Consent Manager が無効化されます。

例:

ConsentPolicy.GDPR
Consent Policy 説明
ConsentPolicy.GDPR GDPR
ConsentPolicy.CCPA CCPA

Dispatchers

Dispatchers (ディスパッチャー) は、データレイヤーからデータを Tealium エンドポイントに送信するモジュールです。現在使用可能なディスパッチャーは次のとおりです。

Dispatcher Name TealiumConfig のリファレンス
Collect Dispatchers.Collect
RemoteCommands Dispatchers.RemoteCommands
TagManagement Dispatchers.TagManagement

少なくとも 1 つのディスパッチャーが必要です。ディスパッチャーが 1 つも指定されていない場合は、トラッキングはなされません。

Expiry

有効期限のあるプロパティの設定に関する有効期限を定義します。

例:

Expiry.session

使用可能な有効期限オプションは次のとおりです。

値 説明
session (default) 現在のアクティブなセッションのライフタイム
forever アプリがインストールされている間は有効期限は決して切れません
untilRestart アプリが再起動するまで

LogLevel

ログ (記録) されている情報の量を制御する logLevel プロパティを設定します。

以下のログレ ベルを利用できます。

値 説明
LogLevel.DEV アプリケーションの進捗を表示する情報イベント
LogLevel.QA アプリケーションをデバッグするために用いられるデバッグ レベルのイベント
LogLevel.PROD 重要なエラーと失敗などのエラー イベント
LogLevel.SILENT ログがありません (デフォルト)

例:

LogLevel.DEV

TealiumEnvironment

環境は、3 つのデフォルト環境 (Dev (開発)、QA、Prod (本番)) のいずれか、または Tealium が公開するカスタム環境です。これらの環境の中から 1 つを選択します。

例:

TealiumEnvironment.dev

値 説明
.dev 開発
.qa QA/UAT
.prod 本番
