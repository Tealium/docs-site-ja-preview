データレイヤー

アプリのライフサイクルを通じてデータレイヤー値を管理する方法を説明します。

仕組み

それぞれのトラッキング コールに設定する代わりに、グローバルなデータレイヤー値を設定するには、 AddToDataLayer() メソッドを使用します。このメソッドで追加されたデータレイヤー値は、ライフサイクル イベントを含む、トラッキング対象の各イベントに含まれます。有効期限パラメータを、値を永続する対象の期間に設定します。

データ管理の詳細は、こちらから参照してください。

使用方法

データレイヤー値を設定するには、データ ディクショナリと有効期限パラメータによって AddToDataLayer() を呼び出します。

AddToDataLayer() および Expiryの詳細は、こちらから参照してください。

tealium.AddToDataLayer(new Dictionary<string, object>(1) {
    {"user_language", "en-EN"}
  },
  Expiry.Forever
);

データレイヤー値を取得するには、取得する値の名前によって GetFromDataLayer() を呼び出します。

string myString = (string)tealium.GetFromDataLayer("user_language");
